### Vim 단축키

i : 현재 위치에 insert  
I : 현재 줄 맨앞에 insert  
a : 현재 커서 다음칸 insert  
A : 현재 줄 맨뒤에 insert  
O : 윗줄 insert  
o : 아랫줄 insert  

w : 단어 첫글자로 이동  
W : 화이트스페이스 단위로 다음 들자로 이동  
gg : 문서 맨 앞으로  
G : 문서 맨 끝으로  

dd : 줄 삭제  
u : undo  
p : 붙여넣기  
y : 복사  
c : 잘라내기  
v : 비쥬얼모드  

일단 자주 쓰는것만.
